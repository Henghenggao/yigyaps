## 2024-03-05 - Enhance Pagination Accessibility
**Learning:** For dynamic content like pagination, it's crucial to use a `<nav>` landmark with an `aria-label` (e.g., `aria-label="Pagination Navigation"`) to provide proper context. Additionally, `aria-live="polite"` should be used on dynamic results counts so screen readers announce changes, and `aria-current="page"` accurately indicates the currently active page. Explicit `aria-label`s on next/prev buttons ensure clarity for users who cannot see the visual arrows.
**Action:** When creating or maintaining list controls or pagination components, always combine `nav` landmarks with `aria-live` regions for status text, use `aria-current` for active items, and add explicit labels to icon-only or vaguely-labelled navigation buttons.
