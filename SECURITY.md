# Security Policy

## Supported Versions

The following versions of YigYaps are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of YigYaps seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to security@yigyaps.com (mock address for now) or contact the maintainers directly through official channels.

When reporting a vulnerability, please include:

- A description of the issue.
- Steps to reproduce the vulnerability (proof of concept).
- Potential impact if exploited.

We will acknowledge receipt of your report within 48 hours and provide a timeline for resolution.
