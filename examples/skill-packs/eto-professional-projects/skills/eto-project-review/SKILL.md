# ETO Project Review

Use this skill when a finance user asks for engineer-to-order project analysis, estimate-to-complete, project margin, delivery risk, or forecast-to-complete analysis.

## Inputs

- Project actual cost and revenue position
- Forecast cost to complete
- Contract value and scope changes
- Known delivery or commercial risks

## Output

- Margin bridge from original budget to current forecast
- Key risk drivers and sensitivity points
- Follow-up checks for variance or risk review skills
