# Yigfinance Shared Protocols Reference

## Finance Safety Rules (G6)
Six non-negotiable rules protecting against hallucinated financial analysis.

## Currency & FX Protocol
- FX variance as a separate driver
- Constant vs current rate treatment
- Budget rate, average rate, period-end rate options

## Output Conventions
- Variance bridge table format (ranked by magnitude)
- Confidence icons: HIGH ■ / MEDIUM ▲ / LOW ○
- Status symbols for assumption labels
- Residual always visible, never distributed
