# Yigfinance: Variance Review

> Evidence-backed variance decomposition for monthly/quarterly business reviews.

## What it does

Variance Review decomposes the difference between actual performance and a comparison basis into identifiable, evidence-backed drivers:

- **Variance bridge** — ranked by magnitude, grouped by category
- **Driver decomposition** — price/volume/mix for revenue; rate/efficiency for costs
- **Driver classification** — structural (persists) vs timing (reverses) vs one-off
- **Accounting separation** — reclasses and top-side journals labeled separately
- **Residual tracking** — unexplained variance shown explicitly, never hidden
- **Management implications** — what should leadership do about each driver

## Key principles

- Every driver claim must cite evidence (source data, user confirmation, or calculation trace)
- Unexplained residuals > 5% are flagged as material — never distributed across drivers
- Confidence levels shown for every conclusion (HIGH / MEDIUM / LOW)
- Accounting motions separated from operating performance
