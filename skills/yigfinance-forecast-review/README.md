# Yigfinance: Forecast Review

> Pressure-test forecasts. Challenge assumptions. Expose fragility. Define downside triggers.

## What it does

Forecast Review is a **challenger skill** — it does not accept forecasts at face value:

- **Assumption extraction** — identifies key revenue, cost, and operational assumptions
- **Assumption register** — documents value, source, owner, trigger, and sensitivity for each
- **Stress testing** — applies driver-specific stress ranges to material assumptions
- **Sensitivity analysis** — tornado-chart ranking by P&L impact
- **Scenario modeling** — Commit, Base, Downside, and Severe Downside cases
- **Cash risk view** — working-capital pressure, peak cash burn, covenant headroom
- **Credibility verdict** — CREDIBLE / CREDIBLE_WITH_RISKS / WEAK / NOT_CREDIBLE

## Key principles

- "Management judgment" is not sufficient basis for material assumptions
- Downside case is mandatory — no exceptions
- Scenarios require coherent assumption sets (not just sensitivity ranges)
- Cash view required for any approval-oriented work
- Optimism bias is challenged, not accepted

## Skill chain position

```
Finance Context Setup → Data Context Check → [Forecast Review] → Management Pack
                                                                → Risk Review
                                                                → Plan CFO Review
```
