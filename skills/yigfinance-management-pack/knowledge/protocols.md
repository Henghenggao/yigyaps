# Yigfinance Shared Protocols Reference

## Analysis Quality Summary (G2)
Multi-dimensional quality dashboard required in every management pack output.
Dimensions: Context, Data Quality, Evidence Coverage, Unexplained Residual, Confidence, Safety Rules, Open Items.
Verdict: READY / READY_WITH_CAVEATS / NOT_READY.

## Finance Safety Rules (G6)
Six non-negotiable rules for financial analysis safety.

## Output Conventions
- Key message formatting (impact → confidence → actionability)
- Risk table format (impact × likelihood)
- Action table format (owner, deadline, priority)
- Cash/working-capital view structure
