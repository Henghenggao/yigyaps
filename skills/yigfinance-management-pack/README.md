# Yigfinance: Management Pack

> Decision-oriented management summary with built-in quality dashboard.

## What it does

Management Pack is the **output layer** of the Yigfinance analysis chain. It does not perform new analysis — it synthesizes upstream results into:

- **Analysis Quality Dashboard** — aggregated confidence, evidence coverage, data quality, open items
- **Key Messages** — max 5-7, ranked by impact, each citing upstream evidence
- **Top Risks** — ranked by impact × likelihood, with mitigation status
- **Required Actions** — prioritized with owner and deadline
- **Cash / Working Capital View** — status and management recommendation
- **Decision Points** — options with evidence-based recommendations

## Key principles

- Never invents findings — everything traces to upstream analysis
- Confidence is the minimum of all upstream levels
- Open items and concerns from upstream are surfaced, never hidden
- Analysis Quality Dashboard is mandatory in every output

## Skill chain position

```
Finance Context Setup → Data Context Check → Variance Review   ─┐
                                           → Forecast Review   ─┤→ [Management Pack]
                                           → Plan CFO Review   ─┘
```

This is typically the **final skill** in an analysis chain.
